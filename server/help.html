Bot Help

Prefix and Syntax

Prefix: $
Example: $help

Syntax

<> - required parameter
[] - optional parameter

Make sure to replace the names with values.

Commands

$ping - Sends the latency from message sending to message response
$av [user mention] - Gets the avatar of the mentioned user if they're in the server. Defaults to the person sending the message.
$picture - Sends a pseudo-random picture from the bot's reserves.
$membercount - Replies with the number of members in the server.
$userinfo [user mention] - Sends an embed with the critical infomation of the user.
$uptime - Sends a link to the site's uptime page.
$electioninfo - Sends an embed with the infomation of all present and upcoming elections.
$settimezone <decimal hours> - Adds (or updates) the user's timezone to the list of timezones.
$addtimezone <decimal hours> - Adds (or updates) the user's timezone to the list of timezones.
$removetimezone - Removes the sending users entry from the timezone database.

Mod-only commands

$approve <user mention> - Gives a mentioned user in the waiting room the Junior Pilot role and sends a welcome message.
$questioning <user mention> - Sends the mentioned user to the questioning room and sends the list of former roles in mod chat.
$addimage <image attachment> - Adds the attached image to the bot's reserves. Do not delete the image or message. Event Managers can run this command.
