About

Hi! My name is Nicolas, and I'm the developer behind the Events bot.

Events Bot is my first node.js adventure, and it has been quite an adventure. I'm hosting the bot on replit.com. The bot has had many updates, and it's open source as well on the github. I'm using the discord.js library to interact with the Discord API. For those of you that care, I'm in high school, so I don't have all the time to make the bot perfect, but I think it's a really good bot right now.
