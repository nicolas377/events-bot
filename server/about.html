About

Hi! My name is Nicolas, and I'm the developer behind the Events bot.

Events Bot was my first node.js adventure, and it was quite an adventure. I'm hosting the bot on replit.com. The bot has had many updates, and it's open source as well on github. I used the discord.js library to interact with the Discord API. This was made in the middle of my freshman year of high school.

Core Features

  • Pictures - Staff members would add pictures of events, and those pictures would be used in certain bot commands.
  • User Info - Fairly self explanatory.
  • Elections - In GeoFS Events, when elections were run, they were run using the code system from the bot along with Google Forms.
  • Timezones - Users could give the bot their GMT offset, and the bot would put it on display for easier scheduling of events.
  • Moderation - Mods could approve users out of the waiting room when they join and send users to the questioning room for being naughty.
